# Projeto Nutricionista\n\n## Estrutura\n- app/\n  - main.py: ponto de entrada do app desktop (Tkinter)\n  - ui/: telas e formulários\n  - database/: acesso SQLite\n  - models/: dataclasses de domínio\n- tests/: testes Playwright (web)\n- .github/workflows/playwright.yml: CI para Playwright\n\n## Como rodar o app desktop (Windows)\n`powershell\npython -m venv .venv\n.\.venv\\Scripts\\activate\npip install -r requirements.txt\npython -m app.main\n`\n\n## Como rodar os testes Playwright\n`powershell\nnpm install\nnpm test            # executa suite\nnpm run report      # abre último relatório\n`\n\n## Observações\n- O banco 
utritionist.db é criado automaticamente no diretório raiz.\n- Campos de CPF e e-mail possuem validação simples; CPF deve ter 11 dígitos.\n- Datas são salvas em formato ISO e timestamps usam CURRENT_TIMESTAMP do SQLite.\n
